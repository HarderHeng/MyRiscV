# MyRiscV 内存架构重新设计

> 版本：v1.0 | 更新日期：2026-05-04

---

## 1. Tang Nano 9K (GW1NR-9C) 资源分析

| 资源 | 数量/容量 | 使用情况 |
|------|-----------|----------|
| BSRAM (18Kbit) | 26 块 | IRAM×8 + DRAM×4 + 预留12块 |
| 片上 Flash | 76KB (FLASH608K) | 程序存储/启动 |
| LUT4 | ~8640 | CPU + 逻辑 |
| FF | ~6480 | 流水线寄存器 |
| PLL | 1 | 可用于时钟管理 |

## 2. 地址空间映射

### 2.1 最终地址映射

```
地址范围                    大小      设备          说明
────────────────────────────────────────────────────────────────
0x0000_0000 ~ 0x0FFF_FFFF  256MB     保留         未来扩展

0x1000_0000 ~ 0x1000_FFFF  64KB      外设空间     
  0x1000_0000 ~ 0x1000_001F  UART0   串口0
  0x1000_0020 ~ 0x1000_003F  UART1   串口1(预留)
  0x1000_0100 ~ 0x1000_01FF  GPIO    GPIO控制器
  0x1000_0200 ~ 0x1000_02FF  SPI0    SPI控制器
  0x1000_0300 ~ 0x1000_03FF  I2C0    I2C控制器
  0x1000_0400 ~ 0x1000_07FF  PWM     PWM控制器
  0x1000_1000 ~ 0x1000_1FFF  CLINT   定时器
  0x1000_2000 ~ 0x1000_2FFF  PLIC    中断控制器
  0x1000_3000 ~ 0x1000_3FFF  SPI Flash XIP  映射片外SPI Flash
  0x1000_4000 ~ 0x1000_FFFF  保留

0x2000_0000 ~ 0x2012_FFFF  76KB      片上 Flash   启动存储
  0x2000_0000: 启动向量(复位后从这里取指)
  0x2000_0100: Bootloader 入口
  0x2000_1000: 用户程序区

0x8000_0000 ~ 0x8000_FFFF  64KB      高速 SRAM
  0x8000_0000 ~ 0x8000_3FFF  16KB    IRAM (I-cache 可缓存)
  0x8000_4000 ~ 0x8000_5FFF  8KB     DRAM (D-cache 可缓存)
  0x8000_6000 ~ 0x8000_6FFF  4KB     保留
  0x8000_7000 ~ 0x8000_7FFF  4KB     启动ROM(只读,复刻Flash前16KB)

0x8001_0000 ~ 0x8FFF_FFFF  224MB-64KB 保留

0xC000_0000 ~ 0xFFFF_FFFF  1GB       PCIe/外部扩展空间(预留)
```

### 2.2 地址译码表

| 地址高位 [31:24] | 区域 | 说明 |
|------------------|------|------|
| 0x00 | 保留 | 未实现 |
| 0x10 | 外设 | peripherals |
| 0x20 | 片上Flash | on-chip flash |
| 0x80 | 高速SRAM |高速缓存区 |
| 0xC0-0xFF | 保留 | 扩展 |

## 3. 高速缓存设计

### 3.1 I-Cache (指令缓存)

```
┌─────────────────────────────────────────────┐
│              I-Cache (4KB, 2-way)           │
├─────────────────────────────────────────────┤
│  Tag Array    │  Data Array (128 lines×32B) │
│  7-bit tag    │  每行 32 字节 (8×32-bit words) │
├─────────────────────────────────────────────┤
│  Hit: 1 cycle  │  Miss: 等待IRAM/Flash      │
│  块大小: 32B   │  2路组相联                 │
│  总容量: 4KB   │  替换策略: LRU             │
└─────────────────────────────────────────────┘
```

**地址映射:**
- Index: `addr[6:2]` (5 bits, 32 lines)
- Tag: `addr[31:7]` (25 bits, 实际使用 7 bits)
- Offset: `addr[1:0]`

**缓存行格式:**
```
┌──────────────────────────────────────┐
│ Valid(1) │ Dirty(1) │ Tag(7) │ Data │
└──────────────────────────────────────┘
```

### 3.2 D-Cache (数据缓存)

```
┌─────────────────────────────────────────────┐
│              D-Cache (4KB, 2-way)           │
├─────────────────────────────────────────────┤
│  Tag Array    │  Data Array (128 lines×32B) │
│  7-bit tag    │  每行 32 字节                │
├─────────────────────────────────────────────┤
│  Write-back   │  Write-allocate             │
│  块大小: 32B  │  2路组相联                 │
└─────────────────────────────────────────────┘
```

## 4. 存储器设计

### 4.1 IRAM (Instruction RAM) - 16KB

```
┌──────────────────────────────────────────────────┐
│            IRAM 16KB (8 × SDPB)                 │
├──────────────────────────────────────────────────┤
│  端口:                                            │
│  - 指令端口: CPU IF (同步读, 1-cycle延迟)         │
│  - 数据端口: CPU MEM (Load/Store)                │
│  - 调试端口: JTAG SBA (通过DM)                   │
├──────────────────────────────────────────────────┤
│  特性:                                           │
│  - 支持 I-Cache 缓存 (可旁路)                   │
│  - 字节/半字/字访问                             │
│  - 同步读, 写优先                               │
└──────────────────────────────────────────────────┘
```

**地址映射:**
- `0x8000_0000 ~ 0x8000_3FFF`
- Bank选择: `addr[11:9]` (8 banks × 2KB)
- 行选择: `addr[8:2]` (512 words per bank)

### 4.2 DRAM (Data RAM) - 8KB

```
┌──────────────────────────────────────────────────┐
│            DRAM 8KB (4 × SDPB)                   │
├──────────────────────────────────────────────────┤
│  端口:                                            │
│  - 数据端口: CPU MEM (Load/Store)               │
│  - 调试端口: JTAG SBA (通过DM)                   │
├──────────────────────────────────────────────────┤
│  特性:                                           │
│  - 支持 D-Cache 缓存 (可旁路)                    │
│  - 字节/半字/字访问                             │
│  - 同步读写                                     │
└──────────────────────────────────────────────────┘
```

**地址映射:**
- `0x8000_4000 ~ 0x8000_5FFF`
- Bank选择: `addr[10:9]` (4 banks × 2KB)
- 行选择: `addr[8:2]` (512 words per bank)

### 4.3 Boot ROM - 4KB

```
┌──────────────────────────────────────────────────┐
│            Boot ROM 4KB (FLASH镜像)               │
├──────────────────────────────────────────────────┤
│  功能:                                           │
│  - 上电后首先从这里执行                           │
│  - 复制 Flash 前 16KB 到 IRAM                  │
│  - 然后跳转到 IRAM 执行                          │
├──────────────────────────────────────────────────┤
│  地址: 0x8000_7000 ~ 0x8000_7FFF               │
└──────────────────────────────────────────────────┘
```

### 4.4 片上 Flash - 76KB

```
┌──────────────────────────────────────────────────┐
│            片上 Flash 76KB (FLASH608K)           │
├──────────────────────────────────────────────────┤
│  地址: 0x2000_0000 ~ 0x2012_FFFF                │
│                                                   │
│  分区:                                            │
│  - 0x2000_0000: 启动向量 (向量表)               │
│  - 0x2000_0100: Bootloader                     │
│  - 0x2000_1000: 用户程序                        │
│  - 0x2001_0000: 用户数据                        │
│                                                   │
│  读方式:                                          │
│  - XIP (Execute In Place): CPU可直接执行         │
│  - 通过 DMA 复制到 IRAM: 更快执行                │
└──────────────────────────────────────────────────┘
```

## 5. 外设设计

### 5.1 地址映射

```
基址              大小    名称          说明
────────────────────────────────────────────────────
0x1000_0000      32B     UART0        串口0
0x1000_0100      256B    GPIO         GPIO控制器
0x1000_0200      256B    SPI0         SPI控制器
0x1000_0300      256B    I2C0         I2C控制器
0x1000_0400      256B    PWM          PWM控制器
0x1000_1000      4KB     CLINT        定时器
0x1000_2000      4KB     PLIC         中断控制器
0x1000_3000      64KB    SPI_XIP      映射外部SPI Flash
```

### 5.2 UART 寄存器

| 偏移 | 名称 | 访问 | 说明 |
|------|------|------|------|
| 0x00 | TXDATA | WO | 发送数据 [7:0] |
| 0x04 | RXDATA | RO | 接收数据 [7:0], [31]=valid |
| 0x08 | STATUS | RO | [0]=TXBusy, [1]=RXValid |
| 0x0C | CONTROL | RW | [0]=TXEnable, [1]=RXEnable |
| 0x10 | DIVISOR | RW | 波特率分频 |

### 5.3 GPIO 寄存器

| 偏移 | 名称 | 访问 | 说明 |
|------|------|------|------|
| 0x00 | DATA | RW | GPIO 数据 [31:0] |
| 0x04 | DIR | RW | 方向控制 [31:0] (1=output) |
| 0x08 | OUT_EN | RW | 输出使能 |

   ┌─────────┐     ┌─────────┐     ┌─────────┐     ┌─────────┐

## 8. 启动流程

```
上电复位
    │
    ▼
┌───────────────────────────────────────┐
│ 1. Boot ROM 执行 (0x8000_7000)        │
│    - 从 Flash 复制 16KB 到 IRAM        │
│    - 初始化栈指针                      │
│    - 跳转到 Flash 0x2000_0100        │
└───────────────────────────────────────┘
    │
    ▼
┌───────────────────────────────────────┐
│ 2. Bootloader (Flash 0x2000_0100)    │
│    - 初始化外设                       │
│    - 配置 PLL (可选)                   │
│    - 初始化 D-Cache/I-Cache           │
│    - 跳转到用户程序                    │
└───────────────────────────────────────┘
    │
    ▼
┌───────────────────────────────────────┐
│ 3. 用户程序 (IRAM 或 Flash XIP)       │
│    - main() 函数                      │
│    - 使用 C 库 (musl-libc)             │
└───────────────────────────────────────┘
```

## 9. 与原设计的对比

| 项目 | 原设计 | 新设计 |
|------|--------|--------|
| 启动地址 | 0x2000_0000 (Flash) | 0x8000_7000 (Boot ROM) |
| I-Cache | 无 | 4KB 2-way |
| D-Cache | 无 | 4KB 2-way |
| 外设地址 | 0x1000_0000 | 0x1000_0000 (不变) |
| IRAM 地址 | 0x8000_0000 | 0x8000_0000 (不变) |
| 总线 | 简单地址译码 | AHB-Lite 矩阵 |
| Flash 访问 | XIP直读 | XIP + DMA复制 |
